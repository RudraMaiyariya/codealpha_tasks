# Iris Flower Classification using Machine Learning

## Overview

The Iris Flower Classification project aims to classify iris flowers into three different species based on their physical measurements. The three species included in the dataset are:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

Machine Learning techniques are used to train a classification model capable of predicting the species of an iris flower based on sepal and petal dimensions.

## Dataset

The dataset contains 150 observations with the following features:

* Sepal Length (cm)
* Sepal Width (cm)
* Petal Length (cm)
* Petal Width (cm)

Target Variable:

* Species
