# Unemployment Analysis with Python

## Overview

This project focuses on analyzing unemployment trends in India using Python. The objective is to understand unemployment patterns across different regions, compare urban and rural unemployment rates, and study the impact of Covid-19 on employment.

## Dataset

The dataset contains information about:

* Region
* Date
* Frequency
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area (Urban/Rural)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook / Kaggle Notebook

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Preprocessing
4. Exploratory Data Analysis (EDA)
5. Regional Analysis
6. Urban vs Rural Comparison
7. Covid-19 Impact Analysis
8. Correlation Analysis
9. Data Visualization
10. Conclusion

## Key Analysis Performed

* Distribution of unemployment rates
* State-wise unemployment comparison
* Monthly unemployment trends
* Year-wise unemployment analysis
* Labour participation analysis
* Urban and rural comparison
* Covid-19 unemployment trends
* Correlation heatmap

## Results

The project successfully identified unemployment trends across different regions and highlighted the economic impact of Covid-19 on employment rates in India.

## Conclusion

The analysis provides meaningful insights into labour market conditions and unemployment patterns. Such analysis can help policymakers and researchers understand employment challenges and make informed decisions.
